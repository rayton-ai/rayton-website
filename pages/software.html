SIMPL ITS

Intelligent Transportation System

SIMPL ITS는 LiDAR와 AI를 결합한 차세대 교통 관제 플랫폼입니다. 교차로에서 99%의 차량 감지 정확도를 달성하며, Virtual Loop 기술로 기존 루프 검지기를 완벽 대체합니다. 도로 공사 없이 소프트웨어만으로 검지 영역을 자유롭게 설정하고 수정할 수 있습니다.

  • ✓ 차량·보행자·자전거 실시간 감지 및 궤적 추적
  • ✓ Virtual Loop — 도로 공사 없이 소프트웨어로 검지 영역 설정/수정
  • ✓ 교통 신호 제어기와 실시간 연동 (Actuation)
  • ✓ 교통량·속도·크기 등 멀티모달 데이터 분석
  • ✓ 사고·역주행·정차 등 돌발 이벤트 즉시 감지
  • ✓ 악천후 (비, 눈, 안개, 야간) 에서도 98%+ 정확도
  • ✓ Failsafe 및 백업 기능으로 무중단 운영 보장
  • ✓ 기존 ITS 인프라 (C-ITS, UTIS, SCATS) 연동 지원

SIMPL ID

Intrusion Detection

SIMPL ID는 LiDAR 기반 지능형 침입 감지 소프트웨어입니다. 고해상도 3D 포인트 클라우드를 실시간 분석하여 크기·형태·이동 패턴에 따라 객체를 정밀 분류하고, 오경보를 획기적으로 줄입니다. 생체 데이터를 수집하지 않아 개인정보 보호 규정을 완벽히 준수합니다.

  • ✓ 실시간 3D 침입 감지 및 즉시 경보 발생
  • ✓ Dynamic ROI — 관심 영역 동적 조정으로 정밀 추적
  • ✓ 사람/차량/동물 객체 자동 분류 — 오경보 최소화
  • ✓ 날씨·조명 무관 24시간 운용 (비, 눈, 안개, 야간)
  • ✓ 개인정보 비수집 — 생체 데이터 없이 감지
  • ✓ VMS/PSIM 등 기존 보안 시스템과 원활한 연동
  • ✓ 다중 센서 통합 — 넓은 구역 커버리지 확장

vs. 카메라 (개인정보 이슈)

vs. 루프 검지기 (유/무만 감지)
